# Kaverilista
Kaverilista kokonaisuuden suunnitelma:

Aloitan tekemällä html tiedostot ja lisään sinne tarvittavan sisällön.

Sen jälkeen aloitan toteuttaa JavaScriptiä ja lisään tarvittavat lisäykset myös html tiedostoon. Samalla tarkastalen että koodi toimii niinkuin pitääkin.

Sitten vielä lisään yksinkertaiset css-tyylittelyt.

Viimeiseksi teen loppu testit että koodi toimii oikein ja kaikki näkyy oikein.
